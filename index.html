🎲 1. Probabilidad — Conceptos Base

La probabilidad mide la certeza de que ocurra un evento. Siempre esta entre 0 y 1.

Probabilidad clasica (Laplace)
P(A) = casos favorables / casos totales
Propiedades fundamentales
Complementario
P(Aᶜ) = 1 − P(A)

Si hay 70% de lluvia, hay 30% de no lluvia.

Union (A o B)
P(A∪B) = P(A) + P(B) − P(A∩B)

Resta la interseccion para no contarla dos veces.

Interseccion (A y B)
P(A∩B) = P(A) · P(B|A)
Mutuamente excluyentes
P(A∩B) = 0 ⇒ P(A∪B) = P(A)+P(B)
🔀 2. Probabilidad Condicional

La probabilidad de B sabiendo que ya ocurrio A.

Ejemplo: “Sabiendo que saque un 6, que probabilidad hay de sacar un 2?”

Formula clave
P(B|A) = P(A∩B) / P(A)
Complementario condicional: P(Bᶜ|A) + P(B|A) = 1
Independencia de eventos

A y B son independientes si el conocimiento de uno no afecta al otro:

P(A∩B) = P(A) · P(B)

Equivalente: P(B|A) = P(B)

Ejemplo bombillas

P(buena | no funciona) = P(buena ∩ no funciona) / P(no funciona)

(250/400) / (350/400) = 250/350 ≈ 0.71
🧮 3. Teorema de Bayes

Permite actualizar probabilidades al obtener nueva informacion.

Teorema de Bayes
P(Aᵢ|B) = P(B|Aᵢ) · P(Aᵢ) / ∑⃘ P(B|A⃘)·P(A⃘)
TerminoNombreSignificado
P(Aᵢ)A prioriLo que sabiamos antes
P(B|Aᵢ)VerosimilitudProb. de observar B si ocurre Aᵢ
P(Aᵢ|B)A posterioriLo que sabemos tras observar B
∑ P(B|A⃘)·P(A⃘)Prob. total de BDenominador normalizador
Probabilidad Total: P(B) = P(B|A₁)·P(A₁) + P(B|A₂)·P(A₂) + … + P(B|Aₙ)·P(Aₙ)
📊 4. Variables Aleatorias
Discreta

Toma valores contables. Los posibles resultados pueden ser contados, aun cuando sean infinitos.

Ejemplos: num. de fallos hasta lanzar un cohete con exito, num. de alumnos que aprueban, num. de taxis que pasan.

Continua

Toma valores en un intervalo real. Cualquier funcion positiva cuya integral en el dominio sea 1 puede ser una funcion de densidad.

Ejemplos: peso, altura, tiempo, diametro de particulas.

Medidas importantes
Media / Esperanza E[X]
E[X] = ∑ xᵢ · P(xᵢ)   (discreta)
E[X] = ∫ x · f(x) dx   (continua)
Varianza y Desviacion tipica
Var(X) = E[X²] − (E[X])²
σ = √Var(X)
Propiedades lineales: E[aX+b] = a·E[X]+b    Var(aX+b) = a²·Var(X)

🎲 5. Distribucion Uniforme Discreta

La v.a. X sigue una distribucion uniforme discreta con parametro n si la probabilidad es la misma para todos los valores x₁, x₂, …, xₙ.

Funcion de probabilidad
P(X = xᵢ | n) = 1/n   i = 1, 2, …, n
E[X] = (1/n) ∑nᵢ₅₁ xᵢ
Var[X] = (1/n) ∑nᵢ₅₁ (xᵢ − E[X])²
Funcion generatriz de momentos
Mẋ(t) = (1/n) ∑nᵢ₅₁ etxᵢ
Caso especial: Cuando xᵢ = i para i = 1, …, n:
E[X] = (n+1)/2     Var[X] = (n²−1)/12
Ejemplo 4.1 — Bombillas

Se coge una bombilla al azar de una caja con bombillas de 40, 60, 75, 100 y 120 watios (n=5).

P(X = xᵢ) = 1/5 = 0.2 para cada valor
E[X] = (40+60+75+100+120)/5 = 395/5 = 79 watios
Var[X] = (1/5)∑(xᵢ−79)²
🤔 6. Distribucion Bernoulli

Un experimento de Bernoulli tiene exactamente dos posibles resultados: exito o fracaso. Lanzar una moneda es el ejemplo tipico.

  • Cada lanzamiento tiene dos posibles resultados: cara o cruz
  • Los lanzamientos son independientes
  • La probabilidad de exito es constante
X ~ Bernoulli(π)
Funcion de probabilidad
P(X=x|π) = πx(1−π)1−x   x = 0, 1
E[X] = π
Var[X] = π(1−π)
Mẋ(t) = πet + (1−π)
Interpretacion: X = 1 si exito (con prob. π), X = 0 si fracaso (con prob. 1−π).
La distribucion Bernoulli es el bloque basico de la Binomial.
🎯 7. Distribucion Binomial

Condiciones de un experimento Binomial:

  • Numero fijo n de experimentos de Bernoulli
  • Probabilidad de exito π constante en cada repeticion
  • Las repeticiones son independientes
  • X = numero de exitos en las n repeticiones
X ~ Bin(n, π)
Funcion de masa de probabilidad
P(X=x|n,π) = C(n,x) · πx · (1−π)n−x   x = 0,1,…,n
E[X] = nπ
Var[X] = nπ(1−π)
Mẋ(t) = (πet+1−π)n

C(n,x) = n! / (x!(n−x)!)  = combinatorio de n sobre x

Ejemplo 4.2 — Moneda trucada

Lanzar una moneda 10 veces. P(cara) = 0.33. Calcular P(X ≥ 6).

X ~ Bin(10, 0.33)
P(X ≥ 6) = ∑10x=6 C(10,x) · 0.33x · 0.6710−x

En R: pbinom(5, 10, 0.33, lower.tail=FALSE) o sum(dbinom(6:10, 10, 0.33))

PDF vs CDF Binomial (Figura 4.1 del libro):
La PDF muestra las probabilidades puntuales P(X=x). La CDF muestra P(X ≤ x) acumulado, creciente de 0 a 1.
📞 8. Distribucion de Poisson

Modela el numero de veces que ocurre un evento en un tiempo o espacio definido. Ejemplos tipicos:

  • Numero de llamadas al 112 entre la 1 a.m. y 2 a.m.
  • Numero de accidentes en un cruce durante 24 horas
  • Numero de errores en una pagina de un libro
  • Numero de taxis que pasan por una calle

Condiciones para un proceso de Poisson:

  • El numero de eventos en intervalos no solapados es independiente
  • La probabilidad de dos o mas eventos en un intervalo muy pequeno es virtualmente cero
  • La probabilidad de exactamente un evento en un intervalo pequeno es proporcional a la longitud del intervalo
X ~ Pois(λ)
Funcion de masa de probabilidad
P(X=x|λ) = λx e−λ / x!   x = 0, 1, 2, …
E[X] = λ
Var[X] = λ
Mẋ(t) = eλ(et−1)
Aproximacion Poisson a Binomial:
Se puede usar Poisson para aproximar Binomial con λ = nπ si se cumple que π ≤ 0.1 y nπ ≤ 5.
Ejemplo 4.3 — Llamadas al 911 (λ = 2 por minuto)

(a) Ninguna llamada en 1 minuto:

P(X=0) = e−2 · 20 / 0! = e−2 ≈ 0.1353

(b) Menos de 5 llamadas en 1 minuto:

P(X < 5) = ∑4x=0 e−2 2x/x!   En R: ppois(4, 2)

(c) Menos de 6 llamadas en 1 hora (λ = 2×60 = 120):

P(X < 6) = ∑5x=0 e−120 120x/x!   En R: ppois(5, 120)
Nota: Cuando λ cambia de escala (de minutos a horas), se multiplica: λhora = 2 × 60 = 120.
□ 9. Distribucion Uniforme Continua

X es uniforme en [a,b] si todos los valores entre a y b son igualmente probables. Usos habituales: generacion de numeros aleatorios y modelado de tiempos de espera.

X ~ Unif(a, b)
Funcion de densidad (pdf)
f(x|a,b) = 1/(b−a)   a ≤ x ≤ b
Funcion de distribucion acumulada (cdf)
F(x) = (x−a)/(b−a)   para a ≤ x ≤ b
E[X] = (b+a)/2
Var[X] = (b−a)²/12
Mẋ(t) = (etb−eta) / t(b−a)
Ejemplo 4.4 — Particulas de aerosol (diametro entre 3 y 5 nm)

X ~ Unif(3, 5). Se pide el volumen medio de las particulas.

E[X] = (3+5)/2 = 4 nanometros
Var[X] = (5−3)²/12 = 4/12 = 1/3 ≈ 0.333

Si el volumen es proporcional a x², entonces E[volumen] = E[X²] = Var[X] + (E[X])² = 1/3 + 16 = 49/3.

Interpretacion grafica (Figura 4.13):
La PDF es un rectangulo horizontal plano. La CDF es una recta ascendente de 0 a 1. Al cambiar a, el rectangulo se desplaza; al cambiar b−a, cambia la altura y la pendiente.
⏪ 10. Distribucion Exponencial

Modela el tiempo de espera hasta el primer evento de un proceso de Poisson. Si X cuenta el numero de eventos en un intervalo (Poisson con media λ), entonces el tiempo W hasta el primer evento sigue una distribucion Exponencial.

X ~ Exp(λ)
Funcion de densidad
f(x|λ) = λ e−λx   si x ≥ 0
Funcion de distribucion acumulada (CDF)
F(x) = 1 − e−λx   x ≥ 0
E[X] = 1/λ
Var[X] = 1/λ²
Mẋ(t) = (1−λ−1t)−1
Propiedad de falta de memoria (sin memoria)

La exponencial es la unica distribucion continua sin memoria. Se usa para modelar tiempos de vida de componentes electronicos.

Propiedad formal
P(X > t₂+t₁ | X > t₁) = P(X > t₂) para todo t₁, t₂ ≥ 0
Demostracion: e−λ(t₂+t₁) = e−λt₂ · e−λt₁

Interpretacion: Si una bombilla lleva funcionando t₁ meses, la probabilidad de que funcione t₂ meses mas es la misma que si fuera nueva.

Relacion Poisson ↔ Exponencial

Dado un proceso de Poisson con media λ eventos por unidad de tiempo:

  • El numero de eventos en un intervalo sigue una distribucion Poisson(λ)
  • El tiempo hasta el primer evento sigue una Exponencial con media 1/λ
Si λ = num. medio de eventos/unidad → tiempo medio de espera = 1/λ
Ejemplo 4.5 — Bombillas con vida media 8 meses (λ = 1/8)

(a) P(3 ≤ X ≤ 12):

P(3≤X≤12) = F(12) − F(3) = (1−e−12/8) − (1−e−3/8) = e−3/8 − e−3/2 ≈ 0.6858 − 0.2231 = 0.4627

(b) Percentil 95 (el valor que acumula el 95% de probabilidad):

F(x) = 0.95 → 1−e−x/8 = 0.95 → x = −8·ln(0.05) ≈ 23.97 meses

En R: qexp(0.95, rate=1/8)

(c) P(X > 25 | X > 10) por falta de memoria:

P(X > 25 | X > 10) = P(X > 15) = e−15/8 ≈ 0.1534
Γ 11. Distribucion Gamma

Generaliza la Exponencial. Modela el tiempo de espera hasta el α-esimo evento de un proceso de Poisson. Util para variables no negativas con distribuciones asimericas.

Ejemplos: tiempo hasta el α-esimo error en un motor, tiempo hasta el α-esimo accidente en un cruce.

Funcion Gamma Γ(α)
Definicion de la funcion Gamma
Γ(α) = ∫0∞ xα−1 e−x dx   (α > 0)

Propiedades importantes:

  • Para α > 0: Γ(α+1) = αΓ(α)   (recurrencia)
  • Para entero positivo n: Γ(n) = (n−1)!   (factorial generalizado)
  • Γ(1/2) = √π
X ~ Γ(α, λ)
Funcion de densidad
f(x|α,λ) = [λα / Γ(α)] · xα−1 e−λx   x ≥ 0
E[X] = α/λ
Var[X] = α/λ²
Mẋ(t) = (1−λ−1t)−α
Relaciones utiles de la distribucion Gamma
CondicionDistribucion resultanteUso
α = 1X ~ Exp(λ)Tiempo hasta el 1er evento
α = n/2, λ = 1/2X ~ χ²(n)Chi-cuadrado con n grados de libertad
α entero positivoDistribucion de ErlangTiempo hasta el α-esimo evento
Muestra de tamano nX̄ ~ Γ(nα, nλ)Distribucion de la media muestral
Ejemplo 4.6 — Restaurante con λ = 3 coches/minuto

(a) P(al menos 5 coches en 120 segundos = 2 minutos):

En 2 minutos: λ·t = 3×2 = 6 coches esperados. X ~ Pois(6).
P(X ≥ 5) = 1 − P(X ≤ 4)   En R: 1 - ppois(4, 6)

(b) P(mas de 1 minuto antes del segundo coche):

Tiempo hasta 2do coche: W ~ Γ(2, 3). P(W > 1) = 1 − P(W ≤ 1)
En R: 1 - pgamma(1, shape=2, rate=3)

(c) Tiempo medio de espera hasta el tercer coche (dado que ya paso uno):

Tiempo adicional para 2 coches mas: W ~ Γ(2, 3). E[W] = α/λ = 2/3 minutos ≈ 40 segundos
🔔 12. Distribucion Normal (Gaussiana)

La distribucion mas importante en estadistica. Muchas poblaciones tienen distribuciones que se aproximan por una Normal. Ademas, la media muestral sigue aproximadamente una distribucion normal (Teorema Central del Limite).

X ~ N(μ, σ)
Funcion de densidad
f(x|μ,σ) = 1/(σ√2π) · e−(x−μ)²/(2σ²)
E[X] = μ
Var[X] = σ²
Mẋ(t) = eμt + σ²t²/2
CDF (no tiene forma cerrada, se usa tabla o software)
F(x) = (1/√2πσ²) ∫−∞x e−(t−μ)²/(2σ²) dt
Propiedades de la campana de Gauss:
- Forma de campana simetrica respecto a la media μ
- Cuanto mayor es σ, mas achatada y ancha es la campana
- El punto de inflexion esta en μ ± σ
- La curva se extiende de −∞ a +∞ (asintota horizontal)
Normal Estandar y Estandarizacion

Una Normal con μ=0 y σ=1, denotada Z, es la Normal estandar. Para calcular probabilidades de cualquier Normal, se estandariza:

Cambio de variable: estandarizacion
Z = (X − μ) / σ   ~ N(0,1)
Calculo de P(a ≤ X ≤ b)
P(a≤X≤b) = P((a−μ)/σ ≤ Z ≤ (b−μ)/σ) = Φ((b−μ)/σ) − Φ((a−μ)/σ)
Notacion zα y cuantiles

zα es el valor de Z que tiene α de su area a la izquierda: P(Z < zα) = α.

Para calcular Xα donde X ~ N(μ, σ):

En R: qnorm(p, mean=μ, sd=σ)
donde p es la probabilidad acumulada a la izquierda

P(a < X < b)Φ(z₂) − Φ(z₁)pnorm(b,μ,σ) − pnorm(a,μ,σ)

Ejemplo 4.25 — Resistencia de componentes electronicos

Resistencia ~ N(μ, σ). Datos: P(X ≤ 0.5) = 0.50 y P(X ≥ 0.628) = 0.10.

(a) Hallar μ y σ:

P(X ≤ 0.5) = 0.50 → la mediana = la media → μ = 0.5 Ω
P(X ≥ 0.628) = 0.10 → P(X ≤ 0.628) = 0.90 → z = qnorm(0.90) ≈ 1.2816
(0.628 − 0.5)/σ = 1.2816 → σ = 0.128/1.2816 ≈ 0.1

(b) P(X < 0.7):

Z = (0.7 − 0.5)/0.1 = 2.0 → P(Z < 2) = 0.9772
En R: pnorm(0.7, 0.5, 0.1)

(c) De 20 componentes, P(al menos 19 con resistencia < 0.7Ω):

p = 0.9772, n = 20, X ~ Bin(20, 0.9772)
P(X ≥ 19) = P(X=19) + P(X=20)
En R: pbinom(18, 20, 0.9772, lower.tail=FALSE)

📈 13. Q-Q Plot (Grafico Cuantil-Cuantil)

El Q-Q plot es uno de los procedimientos graficos mas utiles para verificar si los datos siguen una distribucion normal. Compara los cuantiles de los datos con los cuantiles teoricos de la Normal estandar.

Funciones en R:

  • qqnorm(): calcula los cuantiles de los puntos (i−1/2)/n para n > 10
  • qqline(): traza la linea de referencia conectando los cuantiles (Q1 y Q3)

Como interpretarlo:

  • Si los puntos siguen aproximadamente la linea diagonal → los datos son normales
  • Desviaciones sistematicas en los extremos → colas mas pesadas o ligeras que la Normal
  • Curva en S → asimetria en los datos
Ejemplo (Figura 4.23): Puntuaciones de 20 universitarios. Los puntos siguen la linea diagonal de forma bastante ajustada, lo que indica que los datos son aproximadamente normales, con un ligero alejamiento en el extremo superior.

🔗 14. Distribuciones Multivariadas — Introduccion

En un mismo espacio muestral se puede definir mas de una variable aleatoria.

Ejemplo clasico — lanzar dos dados: X = suma, Y = producto, V = |resta|.

Funcion de Probabilidad Conjunta (discreta)
Definicion
pẋẎ(x,y) = P(X = x, Y = y)

Propiedades:

  • pẋẎ(x,y) ≥ 0 para todo x e y
  • ∑ₓ ∑ₖ pẋẎ(x,y) = 1
  • P[(X,Y) ∈ A] = ∑∑(x,y)∈A pẋẎ(x,y)
Distribuciones Marginales

Se obtienen sumando sobre la otra variable:

pẋ(x) = ∑ₖ pẋẎ(x,y)
suma de fila / total
pẎ(y) = ∑ₓ pẋẎ(x,y)
suma de columna / total
Probabilidad Condicional Multivariada
Probabilidad condicional de X dado Y=y
pẋ|Ẏ(x|y) = pẋẎ(x,y) / pẎ(y)

Ejemplo: P(X=3 | Y=4) = (5/200) / (14/200) = 5/14 ≈ 0.357

⚖️ 15. Independencia de Variables Aleatorias

Las v.a. X e Y son independientes si para todos los posibles valores:

Condicion de independencia
pẋẎ(x,y) = pẋ(x) · pẎ(y)
Importante: Basta con que un solo par (x,y) no cumpla la condicion para concluir que son dependientes.
Ejemplo 1(d) — Comprobacion

Par (X=4, Y=4): nota A en ambas asignaturas.

pẋẎ(4,4) = 2/200 = 0.01
pẋ(4) · pẎ(4) = (21/200) · (14/200) = 294/40000 ≈ 0.00735
0.01 ≠ 0.00735 → X e Y son DEPENDIENTES
📐 16. Covarianza

Mide el grado de dependencia lineal entre X e Y.

Definicion
Cov[X,Y] = E[(X−μₓ)(Y−μₖ)]
Atajo de calculo
Cov[X,Y] = E[X·Y] − E[X]·E[Y]
Cov[X,Y]Interpretacion
> 0Cuando X crece, Y tambien tiende a crecer
< 0Cuando X crece, Y tiende a decrecer
= 0No hay relacion lineal (pueden ser dependientes)
Trampa: Cov=0 NO implica independencia. Ver Ejemplo 2 a continuacion.
Ejemplo 2 — Cov=0 pero dependientes
Y=−1Y=0Y=1pẋ(x)
X=−11/81/81/83/8
X=01/801/82/8
X=11/81/81/83/8
pẎ(y)3/82/83/81
E[X]=0, E[Y]=0, E[XY]=0 → Cov[X,Y]=0
Pero: pẋẎ(1,1)=1/8 ≠ pẋ(1)·pẎ(1)=(3/8)(3/8)=9/64 → DEPENDIENTES
Conclusion: Cov=0 NO implica independencia. Este ejemplo lo demuestra.
📉 17. Coeficiente de Correlacion

Version sin unidades de la covarianza. Siempre entre −1 y +1.

Coeficiente de correlacion de Pearson
ρẋẎ = Cov[X,Y] / (σₓ · σₖ)
ρInterpretacion
−1Relacion lineal negativa perfecta
−1 < ρ < 0Relacion lineal negativa parcial
0Sin relacion lineal (puede haber dependencia no lineal)
0 < ρ < 1Relacion lineal positiva parcial
+1Relacion lineal positiva perfecta
Ejemplo — Dataset 1: μₓ=93, μₖ=100, Cov=280
Var(X) = 9090 − 93² = 441 → σₓ = 21
Var(Y) = 10200 − 100² = 200 → σₖ = √200 ≈ 14.142
ρ = 280 / (21 × 14.142) ≈ 0.943 → Relacion lineal positiva muy fuerte
📋 18. Tablas de Contingencia

Muestran la distribucion conjunta de dos variables discretas. Las celdas interiores son frecuencias conjuntas; los totales de fila y columna son las marginales.

Test χ² de independencia
χ² = ∑ (Oᵢ⃘ − Eᵢ⃘)² / Eᵢ⃘
Eᵢ⃘ = (Total fila i × Total col. j) / Total general

gl = (num. filas − 1) × (num. columnas − 1). Si χ² > χ²critico → variables dependientes.

🚫 21. Errores Tipicos de Examen — No los cometas

Error 1 — Confundir E[X] con E[X²]: Var(X) = E[X²] − (E[X])². Nunca Var(X) = E[X²] directamente.

Error 2 — Cov=0 implica independencia: FALSO. Solo al reves: independencia ⇒ Cov=0.

Error 3 — Olvidar el factorial en Poisson: P(X=k) = λk e−λ / k!. El factorial es obligatorio.

Error 4 — No escalar λ en Poisson: Si λ=2/minuto y preguntan por 1 hora, λhora = 2×60 = 120.

Error 5 — Aplicar sin memoria donde no corresponde: Solo Exponencial (continua) y Geometrica (discreta) tienen la propiedad sin memoria.

Error 6 — No reconocer Gamma(α=1,λ) = Exp(λ): Son la misma distribucion. La Exponencial es un caso especial de la Gamma.

Error 7 — P(X<a) vs P(X≤a) en discretas: En continuas son iguales. En discretas NO: P(X<3) ≠ P(X≤3).

Error 8 — Usar Binomial sin verificar condiciones: n fijo, π constante, ensayos independientes, solo 2 resultados posibles.

Error 9 — Calcular mal el combinatorio: C(n,k) = n! / (k!(n−k)!). Ejemplo: C(5,2) = 120/(2×6) = 10, NO 20.

Error 10 — Dividir celda entre total fila: pẋẎ(x,y) = celda / total general. La marginal pẋ(x) = total fila / total general.
